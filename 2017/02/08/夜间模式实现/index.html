夜间模式实现

发表于 2017-02-08

项目新增夜间模式，实现的方法有很多，在这里记录一下我的实现方法

创建plist文件

command+N –> Property List

该文件我命名为ThemeMode，最终的到ThemeMode.plist文件

然后在plist文件中添加如下内容，key为色值命名，value为对应色值

创建主题管理类

这里我创建一个名为ThemeManager的管理类

直接贴代码

ThemeManager.h文件

#import <Foundation/Foundation.h>

@interface ThemeManager : NSObject

@property (nonatomic, strong) NSDictionary *themeData;

+ (instancetype)sharedThemeManager;

- (void)setThemeMode:(BOOL)isLight;

@end

ThemeManager.m文件

#import "ThemeManager.h"
#import "UserDefaultKeys.h"

@implementation ThemeManager

//读取plist文件
- (NSDictionary *)themeData
{
    if (!_themeData)
    {
        NSString *plistPath = [[NSBundle mainBundle] pathForResource:@"ThemeMode" ofType:@"plist"];
        NSDictionary *data = [[NSDictionary alloc] initWithContentsOfFile:plistPath];
        //获取当前主题下所有key和value
        _themeData = [data objectForKey:[kUserDefault objectForKey:kThemeMode]];
    }

    return _themeData;
}

//单例
+ (instancetype)sharedThemeManager
{
    static ThemeManager *themeManager = nil;
    static dispatch_once_t oncePredicate;
    dispatch_once(&oncePredicate, ^{
        themeManager = [[self alloc] init];
    });

    return themeManager;
}

//设置主题
- (void)setThemeMode:(BOOL)isLight
{
    [kUserDefault setObject:isLight ? @"Light" : @"Dark" forKey:kThemeMode];
    self.themeData = nil;
}

@end

扩展UIColor

在扩展UIColor的文件里，编写方法如

+ (UIColor *)titleColor
{
    //获取对应色值
    NSString *hexValueStr = [[ThemeManager sharedThemeManager].themeData objectForKey:@"Title"];
    return [self colorWithHexColorString:hexValueStr];
}

其中key对应plist文件的key

使用

直接调用UIColor的扩展方法即可
eg.

titleLabel.textColor = [UIColor titleColor];
